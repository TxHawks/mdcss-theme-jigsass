    Style Guide

    This style guide describes the GitHub theme for mdcss and uses its own CSS to generate this theme.

    Overview

    This theme is divided into 3 parts. The markup is found inside main.ejs and contains the framework used to build out the logo, navigation, and main content. The style is found inside assets/style.css and contains the styles you see around you. The script is found inside assets/script.js and uses prism to accomplish syntax highlighting within code blocks.

    Logos

    The theme logo may be configured by setting the logo option.

    require('mdcss')({
    theme: require('mdcss-theme-github')({
        logo: 'path/to/my/logo.svg'
    })
});

    Colors

    The site uses a white #fff background and mine shaft #333 text color.

    Typography

    This theme uses a common Helvetica font stack with a relative line height of 1.6 times the font size.

    Headings

    Headings match GitHub documentation conventions. In this theme, you can add _example to your code blocks to add a living example to your style guide.

    <h1>Heading Level 1</h1>
<h2>Heading Level 2</h2>
<h3>Heading Level 3</h3>
<h4>Heading Level 4</h4>
<h5>Heading Level 5</h5>
<h6>Heading Level 6</h6>

    Iconography

    This site has no icongraphy.

    Layout

    We use flex to horizontally stack the heading and main content areas. The navigation uses fixed position to stay on the screen as you scroll.
